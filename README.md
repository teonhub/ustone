###  Установка и запуск

   ```bash  
   # clone
   git clone https://github.com/teonhub/ustone.git
   npm install
   ```
   ```bash
   # start  
   npm run dev  
   ```
   ```bash
   # open app 
   url: http://localhost:3000
   ```
   ```bash
   # build
   npm run build  
   ```
   ```bash
   # generate ssg  
   npm run generate
   ```
